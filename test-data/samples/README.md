# Sample Files Directory

This directory contains real-world sample files from various legacy systems for comprehensive testing.

## Categories

### Word Processors
- WordStar documents
- AppleWriter files  
- Perfect Writer documents
- MultiMate files

### Computer Systems
- Apple II files
- Commodore 64 documents
- TRS-80 files
- IBM PC formats

### Specialized Formats
- Brother word processors
- Smith Corona devices
- Xerox systems
- Wang computers

## Usage

These samples should be:
1. Small representative files (< 5KB preferred)
2. Non-copyrighted content
3. Well-documented with source system info
4. Paired with expected transformation outputs

## File Naming Convention

`{system}_{model}_{description}.{original_ext}`

Examples:
- `apple2_applewriter_letter.doc`
- `commodore64_speedscript_report.txt`  
- `wordstar_3.3_memo.ws`
